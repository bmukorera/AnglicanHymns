SAMU

SAMU 3

1. TENZI, avo vanonditambudza vakawanda kwazvo: avo vanorwa neni vazhinji.


2. Vazhinji vanotaura nomweya wangu kuti : Haana rubatsiro muna Mwari wake.


3. Asi iwe Tenzi uri mudziviriri wangu : iwe ndiwe wandinoombera, nomusimudzi womusoro wangu.


4. Ndakadaidza kuna Tenzi neizwi rangu : akandinzwa ari mugomo rake rinoyera.


5. Ndakarara ndikabatwa ngehope, ndikamukazve : nokuti Tenzi wakandichengeta.


6. Handingatongotyi gumi ramakumi amazana avanhu : avo vakatenderedza kurwa neni.


7. Simuka, Tenzi, undibatsire, Mwari wangu : nokuti unopamhadza mhandu dzangu dzese mumatama adzo; wakagurura mazino avakaipa.


8. Ruponiso nderwaTenzi : nechikomborero chako chiri panavanhu vako.
