SAMU

SAMU 7

1. TENZI Mwari wangu, ndakasimbira kwauri : ndiponise kune vese vanonditambudza, undisunungure;


2. Nokuti zvimwe ungazoparadze mweya wangu, seshumba, uchiudadzuradadzura : pasina mumwe yo angandibatsire.


3. Tenzi Mwari wangu, kana ini ndakaita chiro chakaipa : kana kuno kunyangara mumaoko angu;


4. Kana ndakaita chakaipa kwaari wakandibata seshamwari : chokwadi, ndakanunura uyo uri mhandu yangu kusina mhosva;


5. Ndizvo mhandu yangu ngaitambudze mweya wangu inditore : hunde, ngaitsiketsikire upenyu hwangu muvhu, nokuradzika kuremedzwa kwangu mumhukuta


6. SIMUKA, Tenzi, mukushatirwa kwako, simuka, pamusana pokushatirwa kwemhandu dzangu : simuka undibatsire mukutonga uko wakatema.


7. Ndizvo sangano ravanhu richakutenderedza : ndizvozvo pamusana pavo simukazve.


8. Tenzi achatonga vanhu; nditonge, Tenzi : sokururama kwangu, nokusava nemhosva kwangu

9. Kunyangara kwavakaipa ngakupere : asi iwe tungamirira vakarurama


10. Nokuti Mwari wakarurama : anoedza pfungwa nemoyo


11. Rubatsiro rwangu runobva kuna Mwari : uyo unoponisa vakarurama mumoyo.


12. MWARI mutongi wakarurama, wakashinga nokutiwa : Mwari anoitwa ashatirwe mazuva ese.


13. Kana munhu asingapinduki, anorodza bakatwa rake : wakunga uta hwake nokuhugadzirisa


14. Wamugadzirira nzanga dzorufu : anogadzirira miseve yake kubaya vanotambudza.


15. Ona, wava napamuviri pokunyangara : wakanamba nhumbu yokusuva, nokubara kuipa.


16. Wakakurira kuchera hunza : akaputsikira mukuparadzwa kwaanga acherera vamwe.


17. Kuva napamuviri kwake kuchauya pamusoro pake : nokunyangara kwake kuchaputsikira pamusoro pomusoro wake.


18. Ndichatenda kuna Tenzi, pamusana pokururama kwake : nokuvirikidza Zita raTenzi Mukuru wevese.
